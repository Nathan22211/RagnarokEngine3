The game can be started by going to Game->Code, and double-clicking on Start.py

To run this game, you will need to install:
+ Python v2.6.5 (http://www.python.org/download/)
+ Pygame v1.9.1 (http://www.pygame.org/download.shtml)

Platforming Block is powered by the Ragnarok Engine
(http://enigmaticfish.blogspot.com/2010/06/ragnarok-engine.html)

*************************************************************
Premise
	You find yourself in a treacherous dungeon and must overcome 
	all the obstacles in order to get home to save your people 
	from a terrible fate.
*************************************************************
Controls													
	Use the left and right arrow keys to control the block	
	Use the up arrow key to jump							
*************************************************************
